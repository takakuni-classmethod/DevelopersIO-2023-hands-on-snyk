# Snyk Code の有効化

Snyk Code を 利用するために Snyk コンソールで Snyk Code の有効化を行います。

1\. 画面左の **Settings** を選択します。

2\. **Snyk Code** を選択し、 **Enable Snyk Code** 欄で **Enabled** をクリックします。

3\. **Save changes** をクリックし設定を保存します。
